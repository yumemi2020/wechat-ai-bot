# 安全政策

## 已知風險

本專案的控制台與 API **預設無登入驗證**。任何能存取後端（`:9950`）或前端（`:9960`）的人，皆可操作微信帳號、AI 設定與對話資料。

**請勿將服務直接暴露於公網。** 建議僅在內網、VPN 或受信任環境中使用；若需對外提供，請自行加上反向代理、HTTPS、IP 白名單或鑑權層。

## 回報漏洞

若發現安全相關問題，請透過 [GitHub Issues](https://github.com/wechat-ai-bot/wechat-ai-bot/issues) 回報。

請在 issue 中盡量描述：

- 受影響的元件與版本
- 重現步驟
- 潛在影響範圍

我們會盡快檢視並回覆。在修補釋出前，請勿公開披露可被利用的細節。
